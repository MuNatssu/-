# 転勤制度とシングルマザー問題の構造的政策案

このリポジトリは、日本の転勤文化と行政制度の不整合が
シングルマザーをはじめとする家庭責任を抱える労働者に
深刻な不利益を与えている構造を分析し、改善策を提示するためのものです。

政策案は docs/ 以下に章ごとに整理されています。
Issue を使って論点を切り出し、Pull Request によって改善を行う
“オープンな政策レビュー環境”として運用します。
