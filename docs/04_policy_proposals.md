# 政策提案：転勤制度と行政制度の再設計

## 1. 転勤の事前登録制
- 労働者が「転勤可否」を事前登録
- シングルマザーは「転勤不可」を選択可能
- 企業は登録範囲内で配置を行う

## 2. 行政の離職理由の見直し
- 子育て・家庭事情による離職を「特定理由離職」に分類
- 失業保険の給付を企業都合と同等に
- 家庭責任を抱える労働者の生活基盤を保護

## 3. 転勤拒否を評価に影響させない制度
- 転勤拒否を理由とした不利益取り扱いを禁止
- キャリア形成の多様化を認める

## 4. 地域定着型雇用の促進
- 地域で働き続けられる雇用モデルの整備
- 保育・学校・地域コミュニティとの連携

## 5. シングルマザー支援制度との統合
- 住宅支援
- 就労支援
- 保育支援
- 転職支援との連携
