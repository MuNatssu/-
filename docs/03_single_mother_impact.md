# シングルマザーへの影響：制度が誘発する貧困リスク

## 1. シングルマザーの脆弱性
シングルマザーは以下の構造的制約を抱える。

- 収入源が一つしかない
- 住居移動が困難
- 保育・学校の継続が不可欠
- 地域コミュニティが生活基盤
- キャリア断絶が即生活困難につながる

## 2. 転勤制度との衝突
転勤命令はシングルマザーにとって致命的。

- 転勤に応じられない → 評価低下
- 拒否 → キャリア停止
- 退職 → 自己都合扱い
- 生活基盤の崩壊

## 3. 行政制度の不整合
行政は家庭事情を「自己責任」と扱うため、

- 失業保険の給付が遅れる
- 生活が不安定化
- 再就職が困難

## 4. 結果としての貧困誘発
これは個人の努力不足ではなく、
**制度が誘発している貧困リスク**である。
