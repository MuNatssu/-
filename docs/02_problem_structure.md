
# 問題の構造分析：前提条件のズレと誘発リスク

## 1. 前提条件のズレ
企業・行政・家庭の三者が、それぞれ異なる前提条件で動いている。

- 企業：昭和モデルの転勤前提
- 行政：家庭事情は「自己責任」
- 家庭：共働き・地域定着が必須

このズレが、構造的なリスクを誘発している。

## 2. 外挿の失敗
過去の成功モデル（高度成長期）を、
現代の前提条件に外挿して適用しているため、制度が破綻している。

例：
- 「転勤すればキャリアアップできる」
- 「家庭はついてくるもの」
- 「単身赴任は可能」

これらは現代では成立しない。

## 3. 誘発される問題
- 転勤拒否 → 評価低下
- 退職 → 自己都合扱い
- 生活基盤の崩壊
- 子どもの教育環境の不安定化
- 家庭責任を抱える人ほど不利になる構造

## 4. 技術者的視点での構造分析
本問題は「個人の努力」ではなく、
**制度設計の前提条件が現実と一致していないことによる構造的故障**である。

予防保全の観点からは、
前提条件の再定義と制度の再設計が必要となる。
