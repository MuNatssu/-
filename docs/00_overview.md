# 転勤制度とシングルマザー問題の構造的政策案（概要）

本政策案は、日本の転勤文化と行政制度の不整合が、
シングルマザーをはじめとする家庭責任を抱える労働者に
深刻な不利益を与えている構造を分析し、改善策を提示するものである。

特に以下の点を重視する。

- 転勤制度が前提とする「昭和モデル」の崩壊
- 行政制度（離職理由分類）が現実と乖離している問題
- シングルマザーが制度的に不利になる構造
- 家庭の安定が労働継続の前提であるという現代的要請
- 企業・行政・家庭の三者の“前提条件のズレ”による誘発リスク

本リポジトリでは、政策案をオープンに議論し、
Issue を通じて論点を分解し、Pull Request によって改善を行う。
